Secure Offline Python-Compatible Workbench v4.2 Visualization Advisor + Session Restore

Single-file HTML. No CDN, no internet, no installation. Runs a practical Python subset through an embedded interpreter with pandas/NumPy/SciPy/Matplotlib-like libraries implemented locally in JavaScript. v4.2 adds an automatic visualization advisor, one-click recommended chart insertion/drawing, expanded chart/graph exports, and portable JSON session save/restore so work can resume later while preserving the broad offline visualization suite: statistical, time-series, risk/Monte Carlo, ML diagnostics, network/graph, map-style, intelligence/security, engineering, financial, dashboard, and 3D-style canvas charts.

Runtime Reality

Honest constraint: this is not full CPython. A plain browser with no local runtime cannot execute native Python packages. This file instead embeds a Python-compatible interpreter for common analytics scripts: functions, imports, variables, loops, if/else, comprehensions, f-strings, dataframe work, statistics, Monte Carlo, and plotting.
